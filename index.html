Typography

A printed work which cannot be read becomes a product without purpose.” – Emil Ruder Typography, A Manual of Design

Font Size

  • Use the default font size for body copy
  • 16px on a monitor equals 12pt in a book
  • Small text causes eye strain
  • Read the text while designing

Text Alignment

  • Left-align body text
  • Fully justified text creates rivers
  • Easier to scan text

Italics

  • Italics can be problematic at smaller sizes
  • Consider using font-weight for emphasis
  • Include italic fonts when using @font-face

Color Contrast

  • Increases readability
  • Smaller text needs higher contrast
  • Too much contrast affects dyslexic users

Demo: contrastrebellion.com

Color Tools

  • PhotoShop (View > Proof Setup > Color Blindness)
  • Color Oracle (colororacle.org)
  • Color Filter (colorfilter.wickline.org)
